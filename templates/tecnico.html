Arquitectura del Sistema

El Software InternadoCBC está construido con una arquitectura modular que permite escalabilidad y mantenimiento eficiente.

Frontend

  • • Framework: React.js
  • • UI Library: Material-UI
  • • Estado: Redux Toolkit
  • • Routing: React Router

Backend

  • • Framework: Node.js + Express
  • • Base de datos: PostgreSQL
  • • ORM: Sequelize
  • • Autenticación: JWT

API Endpoints

Autenticación

POST /api/auth/login

Endpoint para autenticación de usuarios

Gestión de Usuarios
