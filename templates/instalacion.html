Manual de Instalación

Guía paso a paso para instalar el Software InternadoCBC

Requisitos del Sistema

Requisitos Mínimos

  • • Sistema Operativo: Windows 11 Home / Ubuntu 22.04
  • • RAM: 8 GB mínimo
  • • Espacio en disco: 32 GB disponibles
  • • Conexión a Internet

Requisitos Recomendados

  • • Sistema Operativo: Windows 11 Pro / Linux Ubuntu 22.04+
  • • RAM: 16 GB
  • • Espacio en disco: 64 GB disponibles
  • • Conexión a Internet

Proceso de Instalación

1

Descarga del Software

Descarga el instalador desde el repositorio oficial de GitHub del In

2

Ejecutar Instalador

Ejecuta el archivo de instalación como administrador y sigue las instrucciones en pantalla.

3

Configuración Inicial

Configura la conexión a la base de datos y los parámetros iniciales del sistema.

4

Verificación

Verifica que la instalación se completó correctamente ejecutando las pruebas del sistema.
